# 3.1.2 Atur Cara dan Carta Alir bagi Bahasa Penskripan Klien

Semasa membangunkan atur cara penskripan klien, murid melaksanakan proses pengisihan, pencarian, mencari nilai maksimum/minimum, mengira purata, serta penggunaan struktur data seperti giliran.

## 1. Proses Pengisihan (Sorting)
Terdapat dua jenis isihan utama yang dibincangkan iaitu isihan buih (*bubble sort*) dan isihan pilih (*selection sort*). Pengisihan boleh dilakukan secara turutan menaik atau menurun.

### A. Isihan Buih (Bubble Sort)
* **Konsep:** Algoritma mudah yang membandingkan unsur-unsur bersebelahan satu persatu secara berturutan.
* **Proses:** Kedudukan akan ditukar (*swap*) sekiranya berlaku turutan yang salah.
    * Banding unsur pertama dan kedua.
    * Tukar kedudukan jika perlu (contoh: unsur pertama > unsur kedua untuk turutan menaik).
* **Kegunaan:** Sesuai untuk tujuan pembelajaran atau apabila bilangan unsur adalah kecil.

![BubbleSort_Avg_case.gif](https://kalmnest.test/storage/uploads/1766109194_BubbleSort_Avg_case.gif)

```
<html>
<body>
    <script>
        var no = [5,1,4,3,2];
        var bilNombor = no.length, i, j;
        var sementara = no[0];

        for(i=0; i<bilNombor-1; i++) {
            for(j=0; j<bilNombor-i-1; j++) {
                if (no[j] > no[j+1]) {
                    sementara = no[j];
                    no[j]    = no[j+1];
                    no[j+1]  = sementara;
                }
            }
        }

        document.write(no);
    </script>
</body>
</html>
```
### B. Isihan Pilih (Selection Sort)
* **Konsep:** Algoritma yang mencari unsur terkecil atau terbesar dalam senarai.
* **Proses:**
    * Banding satu unsur dengan unsur terkecil/terbesar daripada unsur lain dalam senarai.
    * Tukar kedudukan jika turutan salah.
* **Kegunaan:** Sesuai digunakan sekiranya bilangan unsur adalah kecil (contoh: menyusun pelajar mengikut ketinggian).

![SelectionSort_Avg_case.gif](https://kalmnest.test/storage/uploads/1766109194_SelectionSort_Avg_case.gif)
```
<html>
<body>
    <script>
        var no = [5,1,4,3,2], i, j, min, sementara;
        var bilNombor = no.length;
        for (i = 0; i < bilNombor-1; i++) {
            min = i;
            for (j = i + 1; j < bilNombor; j++) {
                if (no[j] < no[min]) {
                    min = j;
                }
            }
            if (min != i) {
                sementara = no[i];
                no[i] = no[min];
                no[min] = sementara;
            }
        }
        document.write(no);
    </script>
</body>
</html>
```

## 2. Proses Carian (Searching)

### Carian Perduaan (Binary Search)
* **Syarat:** Sesuai digunakan apabila melibatkan bilangan data yang banyak dan data **telah diisih**.
* **Konsep:** Mencari lokasi di tengah-tengah senarai ("Divide and Conquer").
* **Langkah Perbandingan:**
    1. Adakah unsur tengah **sama** dengan unsur yang dicari?
    2. Adakah unsur tengah **lebih kecil** daripada unsur dicari? (Pergi ke sebelah kanan)
    3. Adakah unsur tengah **lebih besar** daripada unsur dicari? (Pergi ke sebelah kiri)


![binary search.gif](https://kalmnest.test/storage/uploads/1766109194_binary%20search.gif)
```
<html>
<body>
    <script>
        var no = [1,2,3,4,5];
        var noCarian = 2;
        var indeksAwal = 0, indeksAkhir = no.length - 1,
            indeksTengah = Math.floor((indeksAkhir + indeksAwal) / 2);
        document.write("No Carian:", noCarian);
        while (no[indeksTengah] != noCarian && indeksAwal < indeksAkhir) {
            if (no[indeksTengah] > noCarian) {
                indeksAkhir = indeksTengah - 1;
                document.write("<br>No tengah yg dirujuk: ",
                    no[indeksTengah], " (Pergi ke sebelah kiri) ");
            } else {
                if (no[indeksTengah] < noCarian) {
                    indeksAwal = indeksTengah + 1;
                    document.write("<br>No tengah yg dirujuk: ",
                        no[indeksTengah], " (Pergi ke sebelah kanan) ");
                }
            }
            indeksTengah = Math.floor((indeksAkhir + indeksAwal)/2);
        }
        if (no[indeksTengah] == noCarian) {
            document.write('<br>No tengah yg dirujuk:', no[indeksTengah], ' (Jumpa)');
        }
    </script>
</body>
</html>

```

## 3. Operasi Statistik & Pengiraan

### A. Nilai Maksimum & Minimum
* **Maksimum (Terbesar):**
    * Beri nilai awal yang rendah (contoh: -1) kepada pemboleh ubah `noMaksimum`.
    * Bandingkan nilai dalam senarai dengan nilai awal. Jika nilai dirujuk lebih besar, kemaskini `noMaksimum`.

```
<html>
<body>
    <script>
        var no = [5,1,4,12,8], noMaksimum = -1,
            i, bilNombor = no.length;
        for(i=0; i<bilNombor; i++) {
            if (no[i] > noMaksimum) {
                noMaksimum = no[i];
            }
        }
        document.write(noMaksimum);
    </script>
</body>
</html>

```
* **Minimum (Terkecil):**
    * Beri nilai awal yang tinggi (contoh: 999) kepada pemboleh ubah `noMinimum`.
    * Bandingkan nilai dalam senarai. Jika nilai dirujuk lebih kecil, kemaskini `noMinimum`.
```
<html>
<body>
    <script>
        var no, pembilang=0, ulang=false, jumlah=0, purata=0;
        do {
            no = parseInt(prompt("Masukkan satu nombor:"));
            pembilang = pembilang + 1;
            jumlah = jumlah + no;
            ulang = confirm("Ulang sekali lagi?");
        } while(ulang == true);

        purata = jumlah/pembilang;
        document.write(purata);
    </script>
</body>
</html>
```
### B. Nilai Mean (Purata)
* **Definisi:** Purata nilai dalam suatu senarai.
* **Formula:** `Mean = Jumlah / Bilangan`.
* **Langkah:**
    1. Jumlahkan nilai-nilai dalam senarai (`jumlah = jumlah + no`).
    2. Dapatkan bilangan nilai (`pembilang = pembilang + 1`).
    3. Bahagikan jumlah dengan bilangan.
```
<html>
<body>
    <script>
        var no, pembilang=0, ulang=false, jumlah=0, purata=0;
        do {
            no = parseInt(prompt("Masukkan satu nombor:"));
            pembilang = pembilang + 1;
            jumlah = jumlah + no;
            ulang = confirm("Ulang sekali lagi?");
        } while(ulang == true);

        purata = jumlah/pembilang;
        document.write(purata);
    </script>
</body>
</html>
```
### C. Nilai Count (Bilangan)
* **Definisi:** Mengira bilangan nilai yang terdapat dalam suatu senarai.
* **Kaedah:** Menggunakan pembilang dengan nilai awal (biasanya 0 atau 1) dan menambah 1 bagi setiap nilai yang dicapai.
```
<html>
<body>
    <script>
        var nama, pembilang = 0,
            ulang = false;
        do {
            nama = prompt("Masukkan nama anda:");
            pembilang = pembilang + 1;
            ulang = confirm("Ulang sekali lagi?");
        } while(ulang == true);

        document.write(pembilang);
    </script>
</body>
</html>
```

## 4. Struktur Data: Giliran (Queue)
* **Definisi:** Struktur data linear (pengumpukan nilai dalam ruang ingatan secara teratur) yang mengamalkan konsep **FIFO (First-In, First-Out)**.
* **Contoh:** Sistem nombor giliran di kaunter bank.
* **Operasi Utama:**
    * **Enqueue:** Memasukkan data ke dalam queue.
    * **Dequeue:** Membuang data lama (terawal) dalam queue.


![queue.gif](https://kalmnest.test/storage/uploads/1766109194_queue.gif)

### Implementasi dalam JavaScript
Menggunakan fungsi tatasusunan (array):

| Kaedah | Masuk Data (Enqueue) | Keluar Data (Dequeue) | Arah Aliran |
| :--- | :--- | :--- | :--- |
| **Kaedah 1** | `push()` (Masuk belakang) | `shift()` (Keluar depan) | Masuk belakang, keluar depan |
| **Kaedah 2** | `unshift()` (Masuk depan) | `pop()` (Keluar belakang) | Masuk depan, keluar belakang |

```
<html>
<body>
    <script>
        var queueNo = [], noBaru, noKeluar;

        noBaru = 5;
        queueNo.push(noBaru);
        document.write("<br>Nombor Baru Masuk: ", noBaru);

        noBaru = -1;
        queueNo.push(noBaru);
        document.write("<br>Nombor Baru Masuk: ", noBaru);

        noBaru = 12;
        queueNo.push(noBaru);
        document.write("<br>Nombor Baru Masuk: ", noBaru);

        document.write("<br><br>Senarai Nombor Terkini Dalam Giliran (Queue):<br>------------------------------------------------");
        document.write("<br>", queueNo);

        noKeluar = queueNo.shift();
        document.write("<br><br>Nombor Dikeluarkan: ", noKeluar);
        document.write("<br><br>Senarai Nombor Terkini Dalam Giliran (Queue):<br>------------------------------------------------");
        document.write("<br>", queueNo);
    </script>
</body>
</html>
```

