#3.3.5 Membina Laman Web Pengguna yang lnteraktif, yang Mengandungi Fitur Pengesahan Data, Butang Pilihan dan Kotak Popup
## **1\. Pengenalan Laman Web Interaktif**

* **Definisi:** Laman web yang membenarkan pengguna berinteraksi dengan kandungan, bukan sekadar membaca.  
* **Aktiviti Interaktif:** Termasuk mengisi borang, meninggalkan komen, menekan butang pilihan, dan memberi respons kepada kotak dialog (*popup*).  
* **Bahasa Penskripan:** Menggunakan **JavaScript** (bahasa penskripan klien) untuk memproses interaksi ini.

---

## **2\. Pengesahan Data (*Data Validation*)**

Proses menyemak data yang dimasukkan pengguna dalam borang sebelum ia diproses.


### **A. Semakan Kekosongan Data**

* Memastikan pengguna tidak membiarkan medan penting kosong.  
* **Contoh:** Jika medan nama kosong, kotak amaran (*alert*) akan muncul: "Sila masukkan nama anda".  
* **Kod:** Menggunakan syarat `if (teksnama == "")`.
```html:src=Pengesahan.html
<html>
  <body>
    Nama Anda:
    <input id="n1"> <button type="button" onclick="semakData()">Pengesahan Nama</button> <script>
      function semakData() // 3
      {
        var teksnama;
        teksnama = document.getElementById("n1").value; // 4
        if (teksnama==" ")
        {
          window.alert("Sila masukkan nama anda"); // 5
        }
      }
    </script>
  </body>
</html>
```

### **B. Semakan Jenis Data (Nombor)**

* Memastikan pengguna memasukkan jenis data yang betul, contohnya nombor giliran.  
* **Fungsi:** Menggunakan `isNaN()` (*is Not a Number*) untuk mengesan jika input bukan nombor.  
* **Contoh:** Jika huruf dimasukkan dalam medan nombor, amaran akan dipaparkan.
```html:src=PengesahanNombor.html
<html>
  <body>
    Nombor Giliran:
    <input id="n2"> <button type="button" onclick="semakNombor()">Pengesahan Nombor Giliran </button>

    <script>
      function semakNombor() // 3
      {
        var teksnombor;
        teksnombor = document.getElementById("n2").value; // 4
        if (isNaN(teksnombor))
        {
          window.alert("Nilai yang dimasukkan adalah bukan nombor. Sila masukkan nombor."); // 5
        }
      }
    </script>
  </body>
</html>
```

## **3\. Butang Pilihan**

Terdapat dua jenis butang utama untuk pengguna membuat pilihan:

### **A. Butang Radio (*Radio Button*)**

* **Fungsi:** Membenarkan pengguna memilih **satu sahaja** pilihan daripada beberapa pilihan (Contoh: Jantina atau Mod Pengajian).  
* **Sifat:**  
  * Menggunakan tag `<input type="radio">`.  
  * Atribut `name` mestilah **sama** bagi semua butang dalam kumpulan itu supaya hanya satu boleh dipilih.  
  * `id` yang unik digunakan untuk rujukan dalam skrip.
```html:src=Radio.html
<html>
  <body>
    <b>MOD PENGAJIAN</b>
    <form action="">
      <input type="radio" name="mod" id="sepenuh">Sepenuh Masa
      <input type="radio" name="mod" id="separuh">Separuh Masa <p>
        <button type="button" onclick="semakMod()">Pengesahan Mod Pengajian </button>
      </p>
    </form>

    <script>
      function semakMod() // 3
      {
        if (document.getElementById('sepenuh').checked == true) // 4
        {
          window.alert("Mod Pengajian: SEPENUH MASA"); // 5
        }
        else
        {
          if (document.getElementById('separuh').checked == true)
          {
            window.alert("Mod Pengajian: SEPARUH MASA"); // 6
          }
        }
        document.getElementById('sepenuh').checked=false;
        document.getElementById('separuh').checked=false;
      }
    </script>
  </body>
</html>
```

### **B. Butang Checkbox**

* **Fungsi:** Membenarkan pengguna memilih **lebih daripada satu** pilihan (Contoh: Subjek yang ingin didaftar).  
* **Sifat:**  
  * Menggunakan tag `<input type="checkbox">`.  
  * Atribut `name` boleh sama untuk kumpulan yang sama.  
  * Pengguna bebas menanda (tick) beberapa kotak serentak.
```html:src=Checkbox.html
<html>
  <body>
    <b>SUBJEK-SUBJEK YANG INGIN DIDAFTAR</b>
    <form action="" name="MyForm">
      <input type="checkbox" name="sub" id="Matematik">Matematik
      <input type="checkbox" name="sub" id="English">English
      <input type="checkbox" name="sub" id="Sains">Sains
      <input type="checkbox" name="sub" id="Bahasa Malaysia">Bahasa Malaysia <p><button type="button" onclick="semakSubjek()">Pengesahan Subjek</button></p> <p><textarea id="SenaraiSubjek" rows="10" cols="30" value="b"></textarea></p>
      <p><button type="button" onclick="resetSubjek()">Reset Subjek</button></p>
    </form>

    <script>
      function semakSubjek() 
      {
        var elLength = document.MyForm.elements.length;
        for (i=0; i<elLength; i++) // 3
        {
          var type = MyForm.elements[i].type;
          if (type=="checkbox" && MyForm.elements[i].checked)
          {
            document.getElementById("SenaraiSubjek").value += 
            MyForm.elements[i].id+'\n';
          }
        }
      }

      function resetSubjek() // 4
      {
        var elLength = document.MyForm.elements.length;
        document.getElementById("SenaraiSubjek").value = " ";
        for (i=0; i<elLength; i++) // 5
        {
          MyForm.elements[i].checked=false;
        }
      }
    </script>
  </body>
</html>
```

## **4\. Kotak Popup (Kotak Dialog)**

JavaScript menyediakan tiga jenis kotak popup untuk interaksi:

### **A. Kotak Alert**

* **Fungsi:** Memberi amaran atau maklumat ringkas kepada pengguna.  
* **Sintaks:** `window.alert("Mesej anda");`.  
* **Contoh:** Memberitahu pengguna "Anda perlu memasukkan nama anda".
```html:src=Alert.html
<html>
  <body>
    Nama:
    <input type="text" > 

    <script>
      alertMasuk(); 
      function alertMasuk()
      {
        window.alert("Anda perlu memasukkan nama anda."); 
      }
    </script>
  </body>
</html>
```

### **B. Kotak Confirm**

* **Fungsi:** Mendapatkan pengesahan daripada pengguna (Ya atau Tidak).  
* **Sintaks:** `window.confirm("Mesej pengesahan");`.  
* **Interaksi:** Mempunyai butang **OK** dan **Cancel**.  
  * Jika **OK**: Mengembalikan nilai `true` (Proses diteruskan).  
  * Jika **Cancel**: Tiada tindakan lanjut.
```html:src=Confirm.html
<html>
  <body>
    Nama:
    <input type="text" > 
    <button onclick="pastiHantar()">Hantar</button> 
    <script>
      function pastiHantar() 
      {
        if (window.confirm("Anda pasti?") == true) 
        {
          document.write("Maklumat telah dihantar."); 
        }
      }
    </script>
  </body>
</html>
```

### **C. Kotak Prompt**

* **Fungsi:** Meminta pengguna memasukkan data (input) sebelum masuk ke laman web.  
* **Sintaks:** `window.prompt("Mesej", "Nilai Default");`.  
* **Interaksi:** Pengguna menaip jawapan dalam kotak dan menekan OK. Data tersebut kemudian boleh dipaparkan dalam laman web (Contoh: "Hai Aminah\!").
```html:src=Prompt.html
<html>
  <body>
    <p>Sila klik butang berikut untuk masukkan nama anda.</p>
    <button onclick="masukNama()">Klik Masuk Nama</button>
    <p id="maklumat"> </p>

    <script>
      function masukNama()
      {
        var pengguna = window.prompt("Sila masukkan nama anda:", "");

        if (pengguna != null)
        {
          document.getElementById("maklumat").innerHTML =
          "Hai " + pengguna + "! Apa khabar hari ini?";
        }
      }
    </script>
  </body>
</html>
```
