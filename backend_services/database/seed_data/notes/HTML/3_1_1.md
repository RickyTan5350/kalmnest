## 3.1.1 Keperluan Bahasa Penskripan Klien dalam Laman Web

### 1. Pengenalan kepada HTML dan CSS
**HTML (Hypertext Markup Language)**
* **Definisi:** Bahasa markup standard yang digunakan oleh pereka web untuk mereka bentuk laman web .
* **Fungsi:** Menentukan struktur dokumen dan bagaimana elemen (teks, audio, video, grafik) disusun agar mudah dibaca .
* **Struktur:**
    * Menggunakan elemen yang terdiri daripada tag berpasangan: `start tag` dan `close tag` .
    * Kandungan web diletakkan di antara tag tersebut .
* **Atribut:** Maklumat tambahan bagi setiap elemen yang terdiri daripada *atribut nama* dan *atribut nilai* .

Contoh tag HTML berpasangan
```
<html>
<head>
    <title>Tajuk Laman Web</title>
</head>
<body>
    Salam Sejahtera
</body>
</html>
```

**CSS (Cascading Style Sheet)**
* **Definisi:** Bahasa pengaturcaraan tambahan (*extension*) kepada HTML .
* **Fungsi:** Membenarkan perubahan gaya (*style*) halaman web dan menerangkan persembahan dokumen (contoh: warna tulisan) .

Contoh HTML tanpa CSS
```
<html>
<head>
    <title>Penggunaan CSS</title>
</head>
<body>
    <h1>Laman Web Sekolah</h1>
</body>
</html>
```
Contoh HTML  dengan CSS
```
<html>
<head>
    <title>Penggunaan CSS</title>
    <style>
        h1 {
            color:blue;
        }
    </style>
</head>
<body>
    <h1>Laman Web Sekolah</h1>
</body>
</html>
```

### 2. Pengenalan kepada JavaScript
* **Definisi:** Bahasa penskripan yang digunakan oleh klien untuk berinteraksi dengan laman web .
* **Fungsi:** Memproses permintaan (*request*) klien dan menjadikan laman web lebih interaktif .
* **Sintaks:** Kod ditulis di antara tag `<script>` dan `</script>` .


**Kaedah Penulisan JavaScript:**
1.  **Explicit Embedding:** Kod JavaScript ditulis terus di dalam atau bersama-sama kod HTML .

HTML tanpa script
```
<html>
<body>
    <p>Nombor:</p>
    5
</body>
</html>
```

HTML beserta script
```
<html>
<body>
    <p>Nombor:</p>
    <script>
        document.write("5");
    </script>
</body>
</html>
```
2.  **Implicit Embedding:** Kod diletakkan dalam fail luaran dengan extension `.js` (contoh: `myScript.js`) dan dipanggil menggunakan atribut `src` .
```
function f1() {
    document.write("5");
}

<html>
<head>
    <title>Script Luaran</title>
    <script src="myScript.js"></script>
</head>
<body>
    <p>Nombor:</p>
    <script type="text/javascript">
        f1();
    </script>
</body>
</html>
```

### 3. Konsep Aplikasi Pelanggan/Pelayan (Client/Server)
Reka bentuk web dibangunkan berteraskan konsep pelanggan dan pelayan .

* **Pelanggan (Client):** Biasanya pelayar web (*browser*) yang digunakan pengguna untuk melayari web .
* **Pelayan (Server):** Bertindak memproses permintaan dan menghantar laman web yang diminta oleh klien .
* **Pangkalan Data (Database):** Komponen untuk menyimpan data .

### 4. Perbezaan Laman Web Statik dan Dinamik
Jadual berikut menunjukkan perbezaan antara kedua-dua jenis laman web :

| Ciri | Laman Web Statik | Laman Web Dinamik |
| :--- | :--- | :--- |
| **Teknologi** | Dibina menggunakan HTML sahaja. | Dibina menggunakan penskripan pelayan (contoh: ASP, PHP). |
| **Kandungan** | Paparan tetap mengikut tag HTML. | Paparan boleh berubah mengikut tindakan pengguna atau data pangkalan data. |
| **Kemaskini** | Lebih sukar (manual). | Mudah dikemas kini oleh pemilik. |
| **Contoh** | Laman web info korporat. | Laman web dengan kalendar aktiviti yang berubah. |

### 5. Jenis Bahasa Penskripan
Penskripan adalah sebahagian kecil program yang dilaksanakan berdasarkan interaksi pengguna .

1.  **Penskripan Klien (*Client-side Scripting*)** :
    * Bekerja di bahagian hadapan (*front-end*).
    * Dilaksanakan oleh **pelayar web (browser)**.
    * Contoh: JavaScript.
2.  **Penskripan Pelayan (*Server-side Scripting*)** :
    * Bekerja di bahagian belakang (*back-end*).
    * Dilaksanakan oleh **pelayan web (server)**.
    * Contoh: PHP, ASP.

**Aliran Proses (Architecture Flow):**
1.  Pengguna menaip URL.
2.  Pelayar menghantar URL ke pelayan.
3.  Pelayan memproses permintaan (penskripan pelayan).
4.  Pelayan memulangkan hasil kepada pelayar.
5.  Pelayar menterjemah HTML dan melaksanakan penskripan klien .

### 6. Bahasa Penskripan Klien
**Fungsi Utama:**
* Memanipulasi dan memaparkan kandungan web yang telah dimuat turun pada komputer klien .
* Memberikan respon pantas terhadap input pengguna .

**Contoh Kegunaan:**
* **Pengesahan Input:** Memeriksa data yang dimasukkan dalam borang dalam talian .
* **Penukaran Elemen:** Menukar jenis font atau warna latar tanpa melibatkan pelayan .
* **Shopping Cart:** Mengira jumlah harga secara automatik apabila item ditambah/dikurangkan .

**Kelebihan:**
* Laman web menjadi lebih interaktif .
* Pelaksanaan tugas yang pantas tanpa penglibatan komputer pelayan .
* Meningkatkan kebolehgunaan web (*usability*) .
* Pembangun web boleh menentukan reka bentuk dan ciri-ciri web .

**Batasan:**
* Tidak boleh melaksanakan tugas yang melibatkan pangkalan data di pelayan .
* Tidak boleh mencapai fail sistem pada pelayan .
