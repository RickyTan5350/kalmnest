# 3.1.3 Atur Cara yang Mempunyai Pemalar, Pemboleh Ubah dan Jenis Data Berlainan

Topik ini membincangkan penggunaan struktur data asas, pemboleh ubah, pemalar, dan pelbagai jenis data dalam atur cara bahasa penskripan klien (JavaScript).

## 1. Asas Pengaturcaraan
* [cite_start]**Pemboleh Ubah (Variable):** Ruang ingatan untuk menyimpan nilai sementara yang boleh berubah-ubah sewaktu atur cara diproses [cite: 1575-1576]. [cite_start]Setiap pemboleh ubah diberi nama pengecam[cite: 1577].
* [cite_start]**Pemalar (Constant):** Ruang ingatan untuk menyimpan nilai sementara yang tidak boleh berubah kerana nilainya telah ditetapkan [cite: 1582-1583]. [cite_start]Dalam JavaScript, kata kunci `const` digunakan[cite: 1587].

## 2. Jenis Data Asas
[cite_start]JavaScript mempunyai beberapa jenis data asas seperti yang ditunjukkan dalam jadual berikut [cite: 1592-1593]:

| Jenis Data | Contoh |
| :--- | :--- |
| **Nombor** | `5`, `100.50` |
| **Rentetan (String)** | `"Ali bin Abu"`, `"BMW 318i"` |
| **Boolean** | `true`, `false` |

## 3. Aplikasi Algoritma dengan Jenis Data Berlainan
Murid dikehendaki melaksanakan proses pengisihan, pencarian, dan struktur data menggunakan jenis data yang berbeza (Nombor dan String).

### A. Isihan Buih (Bubble Sort)
* [cite_start]**Nombor:** Contoh atur cara mengisih bilangan ahli rumah dalam bancian penduduk secara menaik [cite: 1596-1597].
* [cite_start]**String:** Contoh atur cara mengisih senarai nama pesakit klinik mengikut susunan abjad (A-Z)[cite: 1643].

### B. Isihan Pilih (Selection Sort)
* [cite_start]**String:** Contoh atur cara mengisih senarai nama pesakit secara menaik mengikut abjad menggunakan kaedah isihan pilih[cite: 1656].

### C. Carian Perduaan (Binary Search)
* [cite_start]**String:** Contoh atur cara mencari nama pesakit tertentu (contoh: "Ramasamy A/L Muthusamy") daripada senarai yang telah diisih [cite: 1670-1671]. Jika dijumpai, mesej yang sesuai akan dipaparkan.

## 4. Struktur Data: Giliran (Queue)
Dalam bahagian ini, operasi giliran dilaksanakan menggunakan fungsi tatasusunan `unshift()` dan `pop()` untuk memanipulasi data jenis Nombor dan String.

* [cite_start]**Fungsi `unshift()`:** Digunakan untuk memasukkan data ke dalam Queue (dari hadapan)[cite: 1702, 1705].
* [cite_start]**Fungsi `pop()`:** Digunakan untuk mengeluarkan data terawal daripada Queue (dari belakang)[cite: 1687, 1706].

### Contoh Penggunaan:
1.  [cite_start]**Data Nombor:** Memasukkan nombor (5.5, -1.3, 12.95) ke dalam giliran dan mengeluarkan nombor pertama (5.5) [cite: 1686-1687].
2.  [cite_start]**Data String:** Memasukkan nama pesakit ke dalam giliran dan mengeluarkan nama pesakit yang pertama mendaftar [cite: 1705-1706].
