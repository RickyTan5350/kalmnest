### **1\. Pengenalan Atur Cara Interaktif**

Laman web interaktif adalah laman web yang membolehkan interaksi antara pengguna dan sistem.

* **Gabungan Bahasa:** Ia dibina menggunakan **HTML/CSS** untuk struktur/paparan dan **JavaScript** untuk fungsi interaksi.  
* **Kegunaan JavaScript:** Digunakan untuk memanipulasi imej, menambah animasi, serta menyimpan dan memproses data pengguna.

### **2\. Penggunaan Tag Butang (`<button>`)**

Salah satu cara paling asas untuk mewujudkan interaksi adalah melalui penggunaan butang.

* **Tag HTML:** Tag `<button>` digunakan untuk menghasilkan butang fizikal pada paparan laman web.  
* **Sintaks:**  
  HTML

```
<button type="button" onclick="namaFungsi()">Label Butang</button>
```

*   
* **Atribut `onclick`:** Atribut ini bertindak sebagai *event handler*. Apabila pengguna menekan butang, ia akan memanggil dan melaksanakan fungsi JavaScript tertentu.

### **3\. Contoh Aplikasi Interaktif**

#### **A. Aplikasi Pengiraan (Menggunakan Fungsi)**

Contoh 43 dalam dokumen menunjukkan bagaimana butang digunakan untuk memulakan proses pengiraan matematik.

* **Aliran Proses:**  
  1. Pengguna klik butang "Kira".  
  2. Tetingkap `prompt` muncul meminta pengguna memasukkan Nombor 1 dan Nombor 2\.  
  3. Nilai input ditukar kepada jenis nombor menggunakan `parseInt`.  
  4. Fungsi `kira_jumlah(no1, no2)` dipanggil untuk menambah kedua-dua nombor.  
  5. Hasil dipaparkan melalui `alert`.

#### **B. Aplikasi Senarai (Menggunakan Tatasusunan)**

Contoh 44 menunjukkan penggunaan tatasusunan (*array*) dalam atur cara interaktif.

* **Objektif:** Memaparkan senarai peribahasa apabila butang ditekan.  
* **Teknik:**  
  1. Tatasusunan `senaraiPeribahasa` diisytiharkan berisi senarai peribahasa.  
  2. Struktur kawalan ulangan (`for` loop) digunakan untuk membaca setiap item dalam tatasusunan.  
  3. Teks disusun menggunakan simbol `\n` untuk memaparkan setiap peribahasa pada baris baharu dalam tetingkap `alert`.
