### **1\. Pengenalan Subatur Cara (Modul)**

Atur cara JavaScript yang mempunyai barisan arahan yang panjang boleh diringkaskan.

* Arahan-arahan ini boleh dipecahkan mengikut tugas masing-masing dan dihimpunkan dalam blok kod berasingan yang dipanggil **subatur cara** (atau modul)1.  
* Subatur cara utama akan memanggil modul-modul ini untuk dilaksanakan2.  
* Terdapat dua jenis subatur cara iaitu **Prosedur** (*procedure*) dan **Fungsi** (*function*)3.

### **2\. Perbezaan Prosedur dan Fungsi**

Perbezaan utama antara kedua-duanya adalah berkaitan pemulangan nilai (*return value*).

| Ciri | Prosedur (Procedure) | Fungsi (Function) |
| :---- | :---- | :---- |
| **Pemulangan Nilai** | Tidak memulangkan nilai4. | Memulangkan nilai5. |
| **Contoh Definisi** | function toCelcius(f) { ... document.write(...); } | function toCelcius(f) { ... return celcius; } |
| **Cara Panggilan** | toCelcius(fahrenheit); | celsius \= toCelcius(fahrenheit); |
| Nota: Jadual di atas diringkaskan daripada Jadual 3.1.35 dalam dokumen6. |  |  |

### **3\. Cara Memanggil Subatur Cara**

* **Konsep Pemanggil dan Dipanggil:** Subatur cara (pemanggil) akan memanggil nama subatur cara lain (dipanggil) supaya kod di dalamnya dilaksanakan7.  
* **Penggunaan Berulang:** Subatur cara boleh dipanggil berulang kali dengan nilai yang berbeza-beza8.  
  * *Contoh:* Fungsi JumlahNo() boleh digunakan untuk menjumlahkan senarai nombor yang berlainan saiz dan jenis data (seperti integer atau perpuluhan)9999.

### **4\. Parameter**

Parameter ialah pemboleh ubah yang membolehkan nilai dihantar kepada prosedur atau fungsi. Ia ditulis di dalam kurungan ( )10101010.

* **Parameter Sebenar (*Actual Parameter*):** Merujuk kepada pemboleh ubah atau nilai dalam atur cara **pemanggil** yang dihantar keluar11.  
* **Parameter Rasmi (*Formal Parameter*):** Merujuk kepada nama pemboleh ubah dalam subatur cara **yang dipanggil** (fungsi/prosedur) yang akan menerima nilai tersebut12.

**Contoh Aliran Parameter:**

1. **Pemanggil:** Menghantar nilai fahrenheit (cth: 77\) sebagai parameter sebenar13.  
2. **Dipanggil:** Fungsi toCelsius(fahrenheit) menerima nilai 77 tersebut sebagai parameter rasmi14.

### **5\. Memulangkan Nilai (*Return Value*)**

Bagi subatur cara jenis **Fungsi**, ia bukan sahaja menerima kawalan tetapi juga memulangkan nilai hasil pemprosesan semula kepada pemanggil15.

* **Proses:** Fungsi melakukan operasi aritmetik (contoh: pengiraan suhu) dan hasilnya dipulangkan menggunakan arahan return16161616.  
* **Hasil:** Nilai yang dipulangkan itu kemudian boleh dicetak atau digunakan oleh atur cara utama17.
