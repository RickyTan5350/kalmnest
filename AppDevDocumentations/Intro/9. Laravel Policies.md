Clean way to put authorization code

```
php artisan make:policy ChirpPolicy --model=Chirp
```

`app/Policies/ChirpPolicy.php`
```

public function update(User $user, Chirp $chirp): bool
{
    return $chirp->user()->is($user);
}

public function delete(User $user, Chirp $chirp): bool
{
    return $chirp->user()->is($user);
}
```
`$chirp-=>user->is($user)`: checks if chirp user is the user making the request


`ChirpController.php`
```

public function update(Request $request, Chirp $chirp)
{
    if ($request->user()->cannot('update', $chirp)) {
        abort(403);
    }
    // ... rest of method
}
```

or
```

public function update(Request $request, Chirp $chirp)
{
    $this->authorize('update', $chirp);
    // ... rest of method
}
```
`$this->authorize()` checks if user is authorized to delete chirp
